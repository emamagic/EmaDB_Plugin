Config Annotation Inspection

Description

The Config Annotation Inspection is a static code analysis tool designed to validate classes annotated with the @Config annotation in your Java project.

Validation Rules

This inspection checks for the following criteria:

  • @Config must be applied only to classes (not interfaces or enums).
  • Classes annotated with @Config must implement the IConfig interface.

Usage

This inspection helps ensure that your configuration classes conform to the required specifications, promoting better code quality and consistency in the management of configuration settings within your application.
